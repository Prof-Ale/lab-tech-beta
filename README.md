## 🚀 Visão Geral
O **LabTech** é uma solução tecnológica disruptiva que integra os 6 blocos de conhecimento do currículo de Recomposição. O sistema utiliza gamificação e **Inteligência Pedagógica** para oferecer uma jornada personalizada, transformando o erro em uma oportunidade de diagnóstico e intervenção imediata.

## 🧠 Diferenciais Pedagógicos

### 1. Desenho Universal para a Aprendizagem (DUA)
O sistema garante acessibilidade plena através de:
* **Múltiplos Meios de Engajamento:** Narrativa futurista, avatares interativos (ADA) e sistema de progressão por combos.
* **Múltiplos Meios de Representação:** Suporte de áudio (TTS), recursos visuais dinâmicos e linguagem acessível.
* **Múltiplos Meios de Ação:** Interface adaptativa para diferentes ritmos de aprendizagem.

### 2. Motor de Diagnóstico Adaptativo (ADA)
O sistema não apenas corrige; ele **diagnostica**:
* **Clínica do Erro:** Distingue entre falhas de *atenção*, *procedimento* e *conceito*.
* **Risco Cognitivo:** Atribui pesos aos erros (1 a 3), impactando a integridade do sistema (vida) proporcionalmente à gravidade da falha pedagógica.
* **Telemetria BNCC:** Gera relatórios detalhados por habilidade, permitindo ao professor uma visão cirúrgica das lacunas da turma.

## 🛠️ Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3 (Modern UI/UX com variáveis CSS), JavaScript (ES6+ Modules).
* **Gráficos:** Canvas API para representações geométricas e retas numéricas dinâmicas.
* **Acessibilidade:** Web Speech API para narração contextualizada.
* **Persistência:** LocalStorage para preservação do progresso do estudante.

---

## 📊 Impacto na Gestão da Sala de Aula
O LabTech entrega ao professor um **Dashboard de Telemetria** em tempo real, permitindo identificar:
1. Habilidades com menor aproveitamento.
2. Natureza predominante dos erros (Conceitual vs. Operacional).
3. Necessidade de intervenção imediata em grupos específicos.
